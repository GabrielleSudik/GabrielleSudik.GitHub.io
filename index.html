Examples of My Work:

Dancer's Costume Creator

A data-first project creating a dancer's costume shop. Users can browse existing items and add their own. They can even build and save their own full costumes. Skills used: MVC, SQL, HTML5, CSS3, Bootstrap, Javascript

Travel Review Website

A code-first project allowing users to review travel destinations. Users can read existing reviews and write their own. Skills used: MVC, HTML5, CSS3, Bootstrap, Javascript

T Rex / Tricerotops

A console application allowing users to adopt a dinosaur, then work to make it happy, well-fed, and safe to take for a walk. Skills used: C#, OOP

A data-first project creating an online slam book. Users can post questions for their friends and anonymously answer them. Skills used: MVC, HTML5, CSS3, Bootstrap, Javascript, JQuery
